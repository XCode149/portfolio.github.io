Introduction

Apprenez à créer votre propre jeu point-and-click directement dans le navigateur, sans aucun logiciel externe.

Exemple d'interface d'un jeu point-and-click.

Structure HTML

Organisez vos scènes et objets interactifs dans des divs et sections modulaires.

<div class="scene" id="scene1">
  <img src="background.png" class="bg"/>
  <div class="hotspot" data-action="look">...
</div>

Hotspots

Définissez les zones interactives avec des attributs de données pour gérer les actions.

<div class="hotspot" data-action="pickup" data-item="key">Clé</div>

Scènes

Organisez les différentes scènes du jeu avec un système simple de navigation entre elles.

function goToScene(id){
  document.querySelectorAll('.scene').forEach(s => s.style.display='none');
  document.getElementById(id).style.display='block';
}

Inventaire

Créez un inventaire dynamique pour les objets ramassés.

let inventory=[];
function addItem(item){
  inventory.push(item);
  updateInventoryUI();
}

Dialogues

Affichez les dialogues et descriptions en pop-ups ou zones dédiées.

function showDialogue(text){
  const dlg = document.getElementById('dialogue');
  dlg.textContent = text;
  dlg.style.display = 'block';
}

Mini-moteur JavaScript

Combinez scènes, hotspots et inventaire pour rendre le jeu interactif.

// Initialisation simple
const scenes = document.querySelectorAll('.scene');
scenes.forEach(scene => scene.style.display='none');
goToScene('scene1');
